# Dicee Game (JavaScript DOM Practice)

This project was built as part of my Full-Stack Web Development learning journey.

## Concepts Practiced
- Creating an external JavaScript file
- Generating random numbers using Math.random()
- DOM selection using querySelector and getElementsByClassName
- Dynamically updating image sources
- Conditional logic to determine a winner
- Updating page content dynamically

## Features
- Two dice roll randomly on page load
- Dice images update automatically
- Displays:
  - Player 1 Wins
  - Player 2 Wins
  - Draw

## Tech Used
- HTML
- CSS
- JavaScript
